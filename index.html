AMBI

An automated, console-based scheduler to optimize your time management. The #1 thing that college students find the most difficult is time management. We created this app as a way to solve that problem for the busy academic.

Put in any number of tasks and receive an optimized schedule. Supports break preferences and multiple types of tasks. Dynamically regenerate schedules to pick the one you like the most. It does so using a lightweight algorithm that uses assigned weights, i.e. how important you find a task, and prioritizes those tasks while being able to flexibly assign work times to other, less important ones.

Created using Python, with minimum library use. We used the Abstract Class and Mocking frameworks in order to ensure that the project plan worked smoothly. The design uses a model-view -controller pattern, which allows every module group to be tested individually and modified without affecting other segments.

Created with test-driven development and synced programming. I worked on creating tests, bugfixing, project design, and display design. My primary contribution was the controller input loop and handling frontend preference displays.
